Functions in JS

1. built-in - Math.floor()

2. user-defined

1. function decleration

        function fun_name(parameters){
            //code
        }
        fun_name(args);

2. function Expression

        const fun_name  = function(parameters){
            //code
        }
        fun_name(args);

3. Arrow function

        const fun_name = (parameters) => {
            //code
        }
        fun_name(args);
    
        1. function taking parameters and returning value(1,1)
        2. function taking parameters and not returning value(1,0)
        3. function not taking parameters and returning value(0,1)
        4. function not taking parameters and not returning value(0,0)

NaN - not a number
