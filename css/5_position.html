position property:

used to control the position of html elements

        1.postion:static - this is default position of all html elements(we cannot use top, left,bottom  and right properties with this position)
        2. postion: relative - this will position the element from origial position
        3. postion :absolute - this will position the element from nearest positioned ancestor 
        4. postion :fixed - it will fix the html element
        5. postion: sticky - it will stick the html element when we scroll the page
